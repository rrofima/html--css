Trabalhando com links

Para criar uma referencia a hipertexto, ou seja, link de site, usamos a tag a

Acessando o meu link da Hoyolab

É importante saber que essa tag sozinha é recomendada para usar quando o alvo da pesquisa for do seu própio site podendo usar a tag a + target+_self

Mas caso seja um site externo, usamos a tag a + target=_"blank", que tem a funçãp de criar uma nova janela dentro do navegador, + a tag rel="external", que tem a função de ir para fora do seu site própio
