Temos o primeiro contato com o código base HTML, que é a tag ! e uma introdução sobre ele

Parágrafos e quebras

  • tag h1 - h6: Usada para títulos
  • tag p: Usada para dar início a um parágrafos
  • tag hr: Usadar para deixar uma linha horizontal dentro do site
  • tag &.gt e &.gt: Usada para acrescentar semanticamente o símbolo < e > (gt: grather than e lt: less than)

Adicionando emojis e comentários

Para adionarmos um emoji dentro do nosso código, devemos ultilizar o código fonte de um emoji dentro ta tag &#x + o código do emoji, exemplo:

😂 (&#x 1F602)

E para adicionar um comentaário dentro do código, que não fique visível no site, usamos a tag < ! - - , porém, tudo junto, assim ficará uma escrita verde dentro do própio código

Imagens dentro dos sites

De início temos uma breve citação sobre os direitos autorais e como fazer para não ter probleamas com isso,logo temos sobre os tipos de imagens (JPEG, PNG, OGG) e como cada uma delas pode comportar dentro de um site, sendo uma vantagem ou uma desvantagem, depois é falado sobre os tamanhos de imagens e de como eles podem influenciar no resultado e visualização de seu site

Para adiocionar uma uma imagem dentro de um site, primeiramente devemos ter elas em uma pasta no mesmo lugar onde fica o index.html, para assim termos um atalho direto da pasta dentro do VST (Virtual Studio Code), logo usamos a tag img para adicionar o comando para selecionar uma image, dentro da subtag src, apertamos CTRL + spacebar para colocar noosa imagem dentro do site, e na subtag alt, é para dar título a imagem, sendo assim

Nos é apresentado também a ferramente de edição de imagem GIMP, bastante utilizada por muitos progamadores para alterar coisas da imagem, como tamanho, resolução e até mesmo editar a imagem

      Links

      • tag a : usada para criar links
        • subtag target=_blank : usada para criar uma nova janela dentro do navegador
        • subtag rel="external" : usada para direcionar o usuário para uma nova aba
        • subtag rel="next" : usada quando é adicionando links externos
        • subtag download + type="application/*extensão do arquivo"
        • subtag target=_self : usada quando o alvo do link for dentro do própio site
        • subtag rel="prev" : usad para retornar a página anterior dentro do própio site
        • sub tag ../ : usada para retornar a página anterior (inserida antes da pasta do link)
        • subtag real="nofollow" usada para links não confiáveis ou onde não deseja dar seu aval

      Imagens dinamicas

      • tag picture : acompanhada da subtag source media="(max-width)" srcset="tamanho da imagem em px" type="image/formato da imagem" type, utilizada para adicionar uma imagem com certas coisas a mais

      • subtag source : usada para incluir configurações dentro do código da imagem
      • subtag max-width : usada para definir o tamanho da imagem
      • subtag srcset : usada para escolher a imagem
      • subtag alt : imagem flexível
      • subtag type : usada para informar o formato da imagem

      Para fazer com que a imagem se torne flexível, devemos criar um código em ordem crescente, ou seja, o menor em cima e o maior em baixo


      Áudio em HTML

      • tag audio : tem a função de adicionar um áudio dentro do site
      • tag controls : usada para adicionar um controle para p aúdio
          • tag preload : usadapara definir o compotamento do áudio dentro do site
        • auto : baixa todo o áudio simuktaneamente com o site
        • metadada : baixa apenas as informações básicas do áudio, nome, tamanho ect
        • none : só vai baixar as informaçõesdo áudio quando o usuário apertar o play no áudio


        Vídep em HTML
