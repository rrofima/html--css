Principais Formatações

Negrito / Destaque

Para usar o negrito ou o destaque utilizamos a tag B, que é uma forma não semantica, ou seja, sem signifado

Para dar um destaque forte,basta usar a tag STRONG, que daria um destaque, um negrito mais forte


Itálico

para adicionar a formatação em italico, ou seja, aquela palavra tombada para o lado, basta buscar a tag "i" (sem semantica), ou seja frase em itálico ou frase em itálico

Ou para dar enfase num texto, podemos usar a tag EM, ou seja, frase em enfase com semantica (que tem sentido ou significado para o html)

IMPORTANTE

Em caso de desenvolvimento em html, usar as tags "b" para dar destaque a palavra ou negrito e usar a tag "i" para dar a formatação em itálico,já que essas tags não tem uma semantica, somente um sentido visual

> ❗ Agora caso estivem em CSS, deve-se utilizar as tags "strong" para dar destaque ou negrito no trecho desejado, ou "em" caso tenha a intenção de dar a formatação em itálico

TEXTO MARCADO

É possivel criar um texto marcado usando a tag "mark", ou seja texto marcado, ou caso queira selecionar uma cor diferente da cor padrão do html, usamo a mesma tag "mark" + barra de espaço + style/backgroundcolor, exemplo: texto marcado em vermelho

TEXTOS GRANDES E PEQUENOS

A ideia de agora é inserir um texto grande e um pequeno, sendo a tag "big" para criar um texto maior e a tag "small" para criar um texto pequeno

❗ a tag big não é considerada semantica pois e algo mais visual, por isso ela é considerado obsoleta pelo css, enquanto a tag "small" é considerada pelo o W3C um sentido, que tem um sentido, por exemplo em contratos, como uma cláusula de um contrato

TEXTOS EXCLUIDOS

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag "del", ou seja, texto não considerado.

TEXTO INSERIDO

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois, usamos a tag "ins", com semantica, que seria algo como texto com uma enfase maior

É possivel criar textos sublinhados usando a tag "u", uma tag não semantico, que ficaria algo do tipo, um texto não semantico

TEXTOS SOBRESCRTIOS E SUBSCRITOS

Sobrescrito

Para utilizarmos essa função, devemos usar a tag "sup" no lugar onde desejar o texto sobre escrito, exemplo: X7

Subscrito

E para utilizarmos a a função subscrita, colocamos a tag "sub" no lugar onde sera subscrito: exemplo: H2O
