Listas ordenadas

Usamos a tag ol para criarmos uma lista ordenada

,com uma sequencia numérica, e dentro desta tag, usamos a tag li para ter a numeração fa lista
  1. acordar
  2. escovar os dentes
  3. tomar café da manhã
  4. jogar Genshin Impact
  5. perder o 50/50
  6. sair do Genshin Impact
  7. dormir

A tag ol + type permite selecionar o tipo de numeração da lista, e a tar start nos permite selecio0nar qual será a numeração inicial a lista
