Imagens dinamicas

Tente abrir esse site em vários dispositivos ou aumentar ou diminuir o tamanho do seu navegador

Usamos tag source media type para fazer com que a imagem do site se adeque seu tamanho automaticamente demtro do site, dependendo do dispositivo, usamos os tamanho da imagem em ordem crescente, do menor para o maior e de cima para baixo, exemplo

source media=(max-width: *tamanho da imagem + 50)" type srcset+"*foto selecionada*" type="imagem/png"

Brincando com aúdios

botando música ruim para tocar nesse bagui

Para adicionar um audio dentro de um site usamos a tag audio + controls + autoplay


As vezes pode acontecer de o seu navegador nao suportar o tipo de áudio inserido no codigo, como opções temos: MP3, WAV, OGG
Nesse caso, usamos uma espécie de código preventivo para que o navegador consiga reproduzir o audio, sendo ele:
Devemos usar a tag source:src dentro da tag audio

audio
source src"*audio selecionado*" type"audio/*formato do audio*"


Sendo assim,temos essas opções abaixo, caso nenhuma delas funcione, o problema já é com o navegador, mas caso deseje, pode baixar o áudio mo seu dispositivo música muito ruim
audio>
source src="midia/seqsmmp3" type="audio/mpeg">
source src="midia/seqso.mp3.mp3" type="audio/wav">
source src="midia/seqso.mp3.mp3" type="audio/ogg">
/audio>
!! O site busca em forma de cascata como dele deve reproduzir o audio, d cma para baixo

usando a subtag preload, escolhemos como arquivo irá se comportar, sendo Auto,, Metadata, None

A subtag autoplay deve ser utilidada
