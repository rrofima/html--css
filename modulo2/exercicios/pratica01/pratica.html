Para começar, colocarei um fundo em gradiente para essa bosta ficar bonita, usando a tag background-image e usando a tag background-attachment para o fundo fixar fixo na página e não ter problemas estéticos

Para isso, vou criar um item de cada, id, class, pseudo-class, pseudo-ellement e children

Marcações

  • # (id): usado para marcações individuais
  • . (class): usado para mais de uma marcação
  • : (pseudo-class): usado para interacações
  • :: (pseudo-ellement): usado para fazer uma adição dentro da tag classs
  • > (children): tem como a função de sobrepor e fazer interações

Para faazer altecações na tag div, colocamos ela dentro de uma caixa style

  • Para deixar uma do lado da outra : display:inline-block
  • para mudar a cor do fundo : background-color
  • colocar a borda : border px formato cor
  • para deixar o texto alinhado no meio da caixa : text-align center
  • tamanho : width
  • largura : height

tudo isso foi apenas usando a configuração da tag div, e para adiconar a função hover, criamos mais uma configuração dentro da tag style e adicionamos div:hover
