  • display none : serve para ocultar um termo
  • display block : usado para aparecer o termo ocultado pela tag display none
  • > : usado para indicar algo que sobrepõe

    • ou seja, para ocultar um termo e fazer ele aparecer quando se passa o cursor em cima do texto selecionado como principal, fazemos:


    div> texto principal div>
    p> texto escondido p>


    style>

    div > p {
    display : none
    }

    div : hover > p {
    display : block
    }
