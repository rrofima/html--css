Medidas absolutas do font-size

  • cm : centímetro
  • mm : milímetro
  • in : polegada (2,54cm)
  • px : pixel (1 em)
  • pt : ponto
  • pc : paica

Medidas relativas do font-size

  • em : medida relativa ao tamanho do M maiúsculo de uma fonte (16 pixels) (recomendada)
  • ex : medida relativa a altura x de uma fonte
  • rem : relativa ao tamanho da fonte configurada no body do código
  • vw (view width) : tamanho relativo a largura da tela

O tamanho padrão é 16px

  • font-weigth : peso, define a espessura da letra
    • ligther : pequeno
    • normal : tamanho normal
    • bold : grande
    • bolder : muito grande
    • 100-900 (da largura mínima a largura máxima)(400 padrão)

  • width : largura

  • heigth : altura


Shorthand

  • font-family : fonte da letra
  • font-weight : espessura da letra
  • font-size : tamanho da fonte
  • font-style : carcterística da fonte

para criar uma tag com shorthand, usamos as funções das respectivas tags a cima e nessa exata ordem, assim; font: italic bolder 3em 'work Sans', sans-serif
