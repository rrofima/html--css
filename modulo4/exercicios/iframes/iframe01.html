Padrões iframes

O tamonho padrão de um iframe é de 300x150px, assim podemos alterar as propiedades e height e width

podemos configurar um iframe dentro do própio iframe ou dentro de uma css, mas vale lembrar que o que prevalece são as configurações da css

tag Scolling

Usada para definir a rolagem deum iframe

  • auto: deixa a rolagem padrão do iframe
  • yes: deixa a rolagem normalmente
  • no: bloqueia a rolagem (de acordo com os testes, não é todo dispositivo que aceita essa propiedade)

Frameboarder: 0 e 1 apenas, usado para deixar uma borda np iframe
