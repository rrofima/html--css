Sobre iframes

O frameboarder que está dentro do iframe é a opção de deixar uma linha ou não ao redor do iframe.
0 = sem bordas
1 = com bordar
qualquer outro valor diferente de 0 e 1, será considerado como uma borda


O jeito para configurar a borda de um iframe é utilizando as CSS

    Tamanho
  • O tamanho padrão de um iframe é de 300x150px
  • É possivel mudar as dimensões de tamanho e largura, usando a tag heigh e width dentro de um iframe
  • A configuração predominante é a CSS
    tag scolling
  • scolling auto : rolagem padrão (configuração padrão)
  • scolling yes : barra de rolagem mesmo com o conteudo completamente visível
  • scolling no : não permite a rolagem (alguns navegadores não reconhece essa funcão)
