    Confifurações de segurança
  • sandbox="sandbox" : bloqueia todo acesso externo de um iframe, e sendo a "configuração máxima"
  • referrerpolicy="no-referrer" : bloqueia toda a coleta de dados de um usuários
  • sandbox="allow-same-origin" : Limita o iframes apenas a conteúdos que estao dentro do mesmo servidor
  • allow-forms : permite o uso ed formulários dentro de um iframes
  • allow-script : permite scripts
